Penjelasan Git Revert

Saya dari kelompok 1 akan menjelaskan apa itu Git Revert.

Git revert adalah perintah yang digunakan untuk membatalkan perubahan yang telah dilakukan pada repository Git. Perintah ini membuat commit baru yang membatalkan perubahan commit sebelumnya, sehingga riwayat perubahan tetap terjaga.

Kegunaan Git Revert

Perintah git revert biasanya digunakan ketika kita ingin membatalkan perubahan yang telah dilakukan, tetapi tidak ingin menghapus riwayat perubahan tersebut. Dengan begitu, commit lama tetap ada dan bisa dilihat kembali di masa depan.

Cara Menggunakan Git Revert

Untuk menggunakan perintah ini, kita perlu mengetahui hash commit yang ingin dibatalkan. Setelah itu, jalankan perintah berikut:

git revert abc123

Perintah di atas akan membuat commit baru yang membatalkan perubahan dari commit dengan hash abc123.
